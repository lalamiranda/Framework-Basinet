# Framework Basinet

Projeto voltado à análise de sequências genéticas do vírus da dengue (DENV-1 a DENV-4) utilizando arquivos FASTA e Python.

## Objetivo educacional

Aplicar conceitos de bioinformática na análise de sequências genéticas, promovendo a compreensão de dados biológicos por meio de ferramentas computacionais.

## Sobre o projeto

Este projeto é baseado no Framework Basinet, adaptado para análise de dados genéticos.

Utiliza Python para leitura de arquivos FASTA, processamento das sequências e geração de gráficos para interpretação dos resultados.

## Como executar

- Clone o repositório  
- Execute o notebook no Google Colab ou localmente  
- Rode as células em ordem  

## Funcionalidades

- Leitura de sequências FASTA  
- Processamento de dados biológicos  
- Geração de gráficos
